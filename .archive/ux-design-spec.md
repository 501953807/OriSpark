# OriStudio v3 UX 交互设计规范

> **基于**: 9角色评估报告 + 现有设计系统 + 7个模块 v3 设计文档
> **目标**: 解决易用性问题，确保新手 3 分钟内进入可用状态，消除术语障碍，统一交互模式

---

## 一、首次使用引导流程 (Onboarding Wizard)

替换原 5步纯描述性引导，改为 3步有操作结果的引导。选择结果持久化到 `users.creator_type`、`system_settings.onboarding_completed`。

### Step 1: 选择创作者类型

3列x2行卡片网格。v1版本插画师卡片视觉突出（大尺寸 + 闪光渐变边框 + `推荐` 徽章），其余诚实标注能力状态。

选择后系统自动调整: 作品过程阶段列表 / 默认变现路径 / 推荐分发平台(前3) / 产品品类过滤

### Step 2: 导入第一批作品

大面积友好拖拽区域 + 支持格式图标说明 + "跳过，稍后导入"链接。拖拽文件夹时自动按子文件夹名称创建项目组。

### Step 3: 快速上手引导

成功仪式感 + 简化3步路径可视化 (🎨管理作品 → 🛡️保护版权 → 💰开始变现) + "开始使用"按钮 → 进入首页

## 二、空状态页面设计

**总原则**: 空状态 = "下一步做什么" 的标准答案。每个空状态: 图标 + 标题 + 描述 + 主CTA + 可选副CTA + 提示列表。

| 场景 | 图标 | 标题 | CTA |
|------|------|------|-----|
| 空作品库 | 📁 | 还没有作品 | 导入第一件作品 |
| 空仪表盘 | 🚀 | 开始你的创作之旅 | 导入第一件作品 |
| 空产品列表 | 🎨 | 将 IP 变成产品 | 设计第一个产品 |
| 空收入 | 💰 | 收入数据将自动汇总 | 设计产品 / 探索变现 |
| 空存证记录 | 🛡️ | 保护你的作品 | 快速存证 |
| 空监测结果 | 🔍 | 暂无侵权监测结果 | 开启监测 |
| 空IP登记 | 📋 | 暂无IP登记记录 | 登记指引 |
| 空合作伙伴 | 👥 | 暂无合作伙伴 | 添加伙伴 |
| 空通知 | 🔔 | 暂无通知 | — |
| 搜索无结果 | 🔍 | 未找到匹配结果 | 清除筛选 |

## 四、交互模式规范

### 4.1 列表页统一模式
搜索框(debounce 300ms) + 筛选器(类型+状态) + 视图切换(列表/网格) + 批量操作栏(选中≥1出现) + 分页 + 空状态按规范渲染 + 加载Skeleton + hover快捷操作

### 4.2 详情页统一模式
←返回列表 + 操作按钮 + 预览区(40%宽) + 信息面板(60%宽: 基本信息/属性/关联/版本历史) + 底栏CTA

### 4.3 创建/向导流程统一模式
步骤指示器(已完成可回退) + 底部导航(←上一步/下一步→) + 保存草稿(Pinia store, 支持恢复) + 关闭确认Modal

### 4.4 加载状态规范 (永不空白)
- 页面首次: 全页Skeleton
- 列表数据: 8行Skeleton
- 文件导入: 进度条 + "正在导入 12/47"
- AI生成: 打字动画 "AI 正在为你生成..."
- 存证提交: 4步状态流转

### 4.5 错误状态规范
不说"出错了", 说"发生了什么 + 你可以怎么做"。每个错误场景: 具体描述 + 可能原因 + 重试按钮

### 4.6 通知反馈规范
- 操作成功: Toast 顶部居中 绿色 2秒
- 操作失败: Toast 顶部居中 红色 4秒
- 操作进行中: Toast 蓝色+Spinner 直到完成
- 警告: Banner 内容区顶部 手动关闭
- 确认: Modal 页面居中 用户操作
- 后台完成: Sidebar Badge teal圆点

## 六、模块交互一致性审查

7个模块均需满足: 列表统一模板 / 详情统一布局 / 创建向导规范 / 空状态引导 / 加载Skeleton / 错误友好提示 / 成功Toast

各模块特殊交互:
- **M1 创意资产**: 拖拽导入全页触发、批量导入进度列表、标签inline编辑
- **M2 权利保护**: 存证4步可视化、PDF证书内嵌预览、侵权预警颜色分级
- **M3 IP登记**: 首次进入强制免责声明弹窗、类别多选+置信度、律师审核步骤不可跳过
- **M4 商业转化**: 7步向导Canvas预览拖拽+缩放+底色切换、规格自动校验
- **M5 内容分发**: AI文案打字动画+可编辑、多选平台配置
- **M6 经营管理**: 通知未读红点+展开详情、ECharts图表
- **M7 系统**: 设置10分区、深色模式切换
