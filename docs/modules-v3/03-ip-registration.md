## 一、UPL 法律合规声明 (Legal Compliance)

### 1.1 定位声明

**本模块提供的所有信息仅供参考，不构成法律建议。**

本模块不是商标代理机构，不提供法律服务。按照中国《律师法》及相关规定，以下行为明确不属于本模块功能范围:
- 不提供"可注册性评估"或"成功率预测"
- 不推荐单一尼斯分类
- 不自动提交CNIPA申请
- 不提供法律意见或"最佳策略"
- 定价功能为公开费用整理，非"报价"

### 1.2 7项免责声明

以下7项免责声明在系统中多处触发展示:

## 五、智能助手 (5步引导向导)

### Step1: 选择IP类型
4张卡片: ©️著作权 / ®️商标 / 🎨外观设计 / 💡专利
每张显示: 图标+名称+保护对象+适用场景

**免责声明 #1, #2, #3**: 页面底部灰色小字

### Step2: 选择辖区
网格卡片: 国旗+名称+费用参考+预计时长

### Step3: 关联作品自动预填
- 从[1]创意资产中心作品库选择作品
- 点击[🪄自动预填] → 系统从作品数据+用户Profile提取:
  - 作品名称(title) → 申请表作品名称
  - 作者(author_name) → 申请人
  - 创作日期(completion_date) → 创作完成日期
  - 作品简介(synopsis) → 创作说明书
  - 文件信息 → 作品样本
- 显示预填结果(字段→来源→值)，可手动修改
- 完整性评分: 预填后显示%完成度

### Step4: 表单校验
- 必填项检查
- 格式校验(日期/身份证号/统一社会信用代码)
- 文件规格校验(图片尺寸/格式)
- 完整性评分
- 校验结果显示: ✅通过项 / ⚠️警告项 / ❌错误项

### Step5: 导出材料

> ⚠️ **免责声明 #2**: 导出材料中的信息仅供参考，不构成法律建议。请咨询专业商标代理/律师后提交。

- 📄 生成申请表预览(展示预填的所有字段值)
- 📦 材料清单(标记哪些已准备/需手动准备)
- 📥 导出申请材料包(ZIP含申请表+材料清单+官方链接)
- 🔗 直达官方提交平台

### API
- POST /api/ipr/assistant/prefill — 预填
- POST /api/ipr/assistant/validate — 校验
- POST /api/ipr/assistant/generate — 生成申请表
- POST /api/ipr/assistant/export — 导出材料清单

## 六、类别推荐引擎 — UPL合规修订

### 6.1 修订前 vs 修订后

| 维度 | 修订前 | 修订后 |
|------|--------|--------|
| 推荐方式 | 单一分类推荐 | **多分类推荐 + 置信度百分比** |
| 展示格式 | "系统推荐你注册第25类" | "以下类别可能与你的作品相关" |
| 选择数量 | 单选 | **多选(可全选/部分选)** |
| 法律引导 | 无 | **推荐结果底部: "请与律师确认后选择"** |

### 6.2 推荐引擎设计

**置信度计算逻辑**: 基于 TF-IDF 词频匹配算法 — 作品标签/描述中关键词与尼斯分类商品/服务描述的匹配命中数除以总关键词权重，映射为 0-100% 置信度。置信度 < 30% 不返回(使用 Fallback 策略)。

**API响应格式** (与代码对齐):
```json
{
  "recommendations": [
    {"class_no": "16", "class_name": "印刷品、文具", "confidence": 92, "reason": "作品标签[海报/贴纸]匹配第16类"},
    {"class_no": "25", "class_name": "服装、鞋帽", "confidence": 85, "reason": "作品标签[T恤/卫衣]匹配第25类"}
  ],
  "disclaimer": "以上推荐仅供参考，请咨询商标代理/律师后确认选择"
}
```
> 注: 现有代码使用 `priority`(1-5星)字段，需同步改造为 `confidence`(0-100)百分比字段以符合设计文档。

**输入**: 作品标签列表 + 作品描述 + 创作者类型 + 产品品类

**输出**: 多分类推荐列表

```
┌──────────────────────────────────────────────────────┐
│ 根据你的作品 "[应龙] 山海经系列插画"，以下类别可能相关  │
│                                                      │
│ □ 第16类 — 印刷品、文具                  置信度 92%  │
│   适用: 海报、明信片、贴纸、艺术印刷品                │
│ □ 第25类 — 服装、鞋帽                     置信度 85%  │
│   适用: T恤、卫衣、帽子                               │
│ □ 第21类 — 家居用品、陶瓷                  置信度 63%  │
│   适用: 杯子、餐具、装饰品                            │
│ □ 第9类 — 数字产品                        置信度 51%  │
│   适用: 手机壳、数字壁纸、NFT                         │
│ □ 第35类 — 广告、商业管理                  置信度 38%  │
│   适用: 品牌运营、在线零售                             │
│                                                      │
│ ⚠️ 以上推荐仅供参考，请咨询商标代理/律师后确认选择。   │
│ 预估费用: 选1类 ¥300 | 选3类 ¥900 | 选5类 ¥1,500     │
│ [查找附近代理机构] [了解商标注册流程]                  │
└──────────────────────────────────────────────────────┘
```

**关键设计原则**:
- 不可只返回一条推荐 — 最少返回3条
- 必须有置信度百分比 — 量化不确定性
- 必须有"请咨询律师"引导 — 不可跳过
- 分类描述清晰 — 让创作者理解每类的覆盖范围
- **Fallback策略**: 当所有推荐置信度低于30%时，返回"文创常用类别"预置列表(第16/25/21/9/35类)，并标注"以下为文创领域常用类别，与作品关联度较低，建议咨询律师后选择"

### 7.4 关键约束

- **不可绕过**: 前端+后端双重校验。前端表单 disabled 防止误操作，后端 API 层返回 403 错误码 + 提示信息"必须完成律师审核步骤"

- **选(B)提供帮助**: 显示代理机构查询链接，但不推荐特定机构
- **记录用户选择**: 将选择(A/B/C)和时间戳记录到登记记录中
- **数据库字段**: 需在 `IPRegistration` 模型新增 `lawyer_consulted`(VARCHAR: A/B/C) 和 `disclaimer_accepted_at`(DATETIME) 字段，并创建对应数据库迁移

## 八、商标查询增强 (新增 v1)

### 8.1 功能说明

在 IP 登记模块中增加商标查询能力，帮助用户在提交商标申请前了解已有注册情况。
**不重复造轮子**：通过 Gateway ABC 模式对接第三方商标数据库。

### 8.2 数据模型

```sql
-- 扩展 ipr_registrations 表
ALTER TABLE ipr_registrations ADD COLUMN trademark_query_done BOOLEAN DEFAULT 0;
ALTER TABLE ipr_registrations ADD COLUMN trademark_query_date DATETIME;
ALTER TABLE ipr_registrations ADD COLUMN trademark_query_results JSON;
-- trademark_query_results: [{"name":"XXX","status":"registered","class":"25","date":"2025-01-01","similarity":85}, ...]

-- 新增：商标查询记录表
CREATE TABLE IF NOT EXISTS trademark_queries (
    id TEXT PRIMARY KEY,
    user_id TEXT NOT NULL,
    query_text TEXT NOT NULL,
    jurisdiction TEXT NOT NULL,  -- 'cn' / 'us' / 'eu' / 'global'
    class_no TEXT,              -- 尼斯分类号
    results JSON,               -- 查询结果
    source TEXT DEFAULT 'gateway', -- 'cnipa' / 'uspto' / 'wipo' / 'euipo'
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
CREATE INDEX IF NOT EXISTS idx_tm_query_user ON trademark_queries(user_id);
```

## 十、IP资产仪表盘

### 功能
- 统计卡片: 按IP类型(著作权/商标/外观/专利) × 按状态(已注册/处理中/草稿)
- 总IP数 + 已注册数 + 处理中数 + 年度费用预估
- 辖区分布: 🇨🇳×5 🇺🇸×2 🇪🇺×1 等
- ⏰ 续展提醒: 到期前红色(30天)/橙色(90天)/绿色(>90天)

### API
- GET /api/ipr/portfolio — IP资产汇总
- GET /api/ipr/reminders — 续展提醒列表

## 十二、前端实现

### IprView.vue Tab结构

| # | Tab | 内容 |
|---|-----|------|
| 1 | 📖 登记指引 | 辖区选择器 + 按IP类型的指引卡片 + 底部免责声明 #2, #7 |
| 2 | 🪄 智能助手 | 5步向导 + 首次进入全屏免责弹窗(#1, #2, #3, #7) + Step5 CNIPA律师审核步骤 |
| 3 | 📋 登记记录 | 记录列表+筛选+添加/编辑弹窗 |
| 4 | 📊 IP资产 | 统计卡片+辖区分布+续展提醒 |
| 5 | 💰 费用计算 | 费用计算器表单+结果展示+费用参考说明 |

### 关键交互约束

- **首次进入IP登记模块**: 弹出全屏免责声明弹窗(#1, #2, #3, #7)，必须点击"我已阅读并理解"才能进入
- **类别推荐结果**: 展示多条推荐+置信度，不可只展示一条；底部必须有"请咨询律师"引导
- **CNIPA提交前**: 强制律师审核步骤，三个选项必须选一个，(C)选项触发二次确认
- **免责声明Banner**: 黄色警告框，不可折叠，在类别推荐/费用计算/材料导出等关键节点展示
