1. 设计理念

一句话定义：一个在个人电脑上运行的创作者全链路助手工具，帮助独立创作者快速完成作品管理、存证确权、侵权监测、知识产权登记、产品变现和标准化输出的完整业务闭环。v1 聚焦插画师/AIGC 艺术家，其他创作者类型标注"规划中"。

核心原则

原则说明设计体现
创作者主权数据永远在创作者的电脑上本地SQLite + 本地存储，不上传创作者原始文件
不重复造轮子存证/商标/搜索通过API接入现有平台，电商通过标准协议对接ERPMCP Server + JSON Feed 标准输出，不做重复的电商发布工程
渐进式复杂度新手只看基础功能，高级功能可展开默认界面核心流程，高级设置在设置页
离线优先无网络时仍可使用核心功能本地哈希+本地数据库完全离线可用
全球化创作者的事业无国界 (v1聚焦插画师)WIPO/EUIPO/USPTO/CNIPA 全路径覆盖
AI原生AI代劳创作者不擅长的商业/法律/运营事务自动标签、AI描述、智能类别推荐、自动表单预填

5. 作品管理模块 v3

模块定位：创作者所有数字资产的统一管理中心。面向 AI 新时代创意爆发时代，为个人创作者提供从资产保护到价值变现的全链路管理能力。

5.1 核心功能

三级导入模式（隐私保护）

模式信任等级说明适用场景
🛡️ 仅哈希 (Hash-Only)★☆☆☆不上传文件，仅在本地内存中计算SHA-256后丢弃文件。可存证，不可预览，文件自管高价值商业作品、未公开独家作品
📷 低清预览 (Low-Res)★★☆☆上传缩略图(≤400px)+完整元数据；不保存原始文件。可存证，可低清预览半公开作品、需分类但不想暴露原始分辨率
📦 完整导入 (Full)★★★★上传完整文件，全套功能：预览/编辑/版本管理/批量操作/自动备份信任本地系统，需要完整的全链路功能

过程阶段管理（按文件类型）

文件类型创作生命周期阶段
🖼️ 图片/插画/设计灵感收集(Inspiration) → 草图(Sketch) → 线稿(LineArt) → 上色(Coloring) → 细节(Detail) → 终稿(Final) → 导出变体(Exports)
🎵 音频/音乐灵感片段(Idea) → 编曲(Arrangement) → 录音(Recording) → 混音(Mixing) → 母带(Mastering) → 发行(Release)

📝 文档/文字大纲(Outline) → 初稿(Draft) → 修订(Revision) → 终稿(Final) → 排版(Formatting) → 发布(Publish)
💻 代码原型(Prototype) → 开发(Dev) → 测试(Test) → 审查(Review) → 发布(Release) → 热修(Hotfix)

6. 存证确权模块 v3

模块定位：为个人创作者的数字化作品提供可信、可追溯、可验证、可举证的版权证据固化服务。基于竞品研究(OriginStamp/CZone/至信链/蚂蚁链/Creator Credentials/Liccium)，融合国际标准(W3C DID/VC、C2PA 1.3+、RFC 3161、伯尔尼公约)，构建四层递进式存证体系。

6.1 竞品分析

竞品核心能力启示
OriginStampMerkle树批量聚合→哈希锚定BTC/ETH；多区块链支持；免费层批量锚定降低费用；独立Merkle Proof验证
CZone (LutinX)即时区块链公证+KYC身份绑定+175国法律认可身份绑定增加证书可信度
至信链 (腾讯)互联网法院直连；多地法院采信；企业级SLA司法直连是最强背书
Creator Credentials (EU)W3C DIDs + VCs；符合eIDAS法规；隐私保护国际标准可验证凭证
LicciumISCC内容标识+Verifiable Creator Credentials内容标识标准化方向
蚂蚁链阿里生态；市场份额~26.5%；长三角司法链商业生态整合
保全网/数秦科技杭州互联网法院首例区块链采信案例司法先例参考

6.2 四层存证体系

L1 · 本地时间戳层 (Local Timestamp) 技术: SHA-256分块哈希 + RFC 3161可信时间戳 + ECDSA本地自签名 可信度: ★☆☆☆☆ (个人声明级) 成本: ¥0 · 时间: 秒级 · 离线可用 L2 · 区块链锚定层 (Blockchain Anchoring) 技术: Merkle树批量聚合(OriginStamp模式) → 根哈希锚定ETH/SOL/FIL多链 可信度: ★★★☆☆ (技术可信级) 成本: ¥0.05-0.5/批次 · 时间: 分钟级 特色: 独立Merkle Proof验证(不依赖平台) L3 · 司法平台层 (Judicial Notarization) 平台: 版权家(DCI国家版权局) / 至信链(司法链) / 蚂蚁链(商业链) 可信度: ★★★★★ (司法采信级) 成本: ¥0.5-5/次 · 时间: 分钟到小时级 特点: 互联网法院采信率98%+ L4 · 内容凭证层 (Content Credentials) 技术: C2PA 1.3+ Manifest嵌入 + W3C DID Creator Identity + Verifiable Credentials 可信度: ★★★★☆ (国际标准级) 成本: ¥0 · 时间: 秒级 特点: 纯Python实现(零c2patool依赖) · 符合eIDAS法规

6.3 存证流程

选择作品 → 本地SHA-256哈希 → L1本地时间戳+签名(必选) → L2 Merkle树批量锚定(可选) → L3选择司法平台(版权家/至信链/蚂蚁链)→用户扫码付费→平台确认→生成PDF 2.0证书(含各层证据记录+QR码) → L4 C2PA嵌入(可选)

6.4 AI存证推荐

根据作品价值/类型/预算，自动推荐最佳存证组合方案。高价值商业作品 → L1+L2+L3+L4四层全栈保护；个人练习作品 → L1本地存证即可。
