# OriStudio 用户手册

OriStudio 是一站式创作者工具平台，帮助独立创作者完成从**创作**、**存证**、**变现**到**监控**的全流程管理。

## 1. 快速开始

### 1.1 环境要求

| 组件 | 最低版本 | 推荐版本 |
|------|---------|---------|
| Python | 3.10+ | 3.12+ |
| Node.js | 18+ | 20+ |
| Ollama (可选) | 0.1.0 | 最新版 |
| 浏览器 | Chrome 100+ / Safari 16+ | 最新版 |

### 1.2 安装与启动

**后端启动：**

```bash
cd backend
python3 -m venv venv
source venv/bin/activate    # macOS/Linux
# venv\Scripts\activate     # Windows
pip install -r requirements.txt
python3 -m app.main
```

服务默认运行在 `http://localhost:8765`，API 文档在 `http://localhost:8765/docs`。

**前端开发模式：**

```bash
cd frontend
npm install
npm run dev
```

前端开发服务器默认运行在 `http://localhost:5173`。

**一键启动（推荐）：**

```bash
# 终端1 - 后端
cd backend && python3 -m app.main

# 终端2 - 前端
cd frontend && npm run dev
```

### 1.3 初始配置

首次启动后，访问 `http://localhost:5173/settings` 进行基本配置：

1. **常规设置**: 站点名称、语言、时区
2. **SMTP 邮件**: 用于通知发送和邮箱验证（可选）
3. **AI 设置**: 配置 Ollama 地址（启用 AI 功能需要）

## 3. 创作工作区

### 3.1 上传作品

1. 点击侧边栏「创作」进入工作区
2. 点击「+ 新作品」按钮
3. 选择文件（支持图片、PSD、AI、SVG、PDF 等格式）
4. 填写标题、描述、标签
5. 点击「保存」

### 3.2 作品管理

- **编辑**: 点击作品卡片进入详情，可修改标题/描述/标签
- **删除**: 在作品详情页底部点击「删除」（危险操作）
- **搜索**: 顶部搜索栏支持按标题/标签模糊搜索
- **筛选**: 按文件类型、状态筛选作品列表

### 3.3 支持的文件格式

| 类别 | 格式 |
|------|------|
| 位图 | PNG, JPG, JPEG, GIF, BMP, WEBP |
| 矢量 | SVG, AI, EPS |
| 设计源文件 | PSD, XD, FIG, SKETCH |
| 3D | OBJ, STL, GLB, GLTF |
| 文档 | PDF |

## 4. 版权存证

### 4.1 什么是版权存证

版权存证将您的作品哈希值记录到区块链上，生成不可篡改的时间戳证明。这不等同于法律版权登记，但提供了强有力的原创时间证据。

### 4.2 存证流程

1. 在创作工作区选择一个作品
2. 点击「存证」按钮
3. 选择存证平台：
   - **蚂蚁链**: 国内最大区块链存证平台
   - **版权家**: 专业的版权服务
   - **知信链**: 知识产业联盟链
4. 确认后等待区块链确认（通常几秒到几分钟）
5. 完成后可下载存证证书

### 4.3 查看存证记录

- 点击「存证」菜单查看所有存证记录
- 每条记录包含：存证编号、区块链交易哈希、时间戳、平台
- 点击记录可查看证书详情和验证链接

## 5. 产品数据枢纽 (PDH)

### 5.1 创建商品

1. 进入「变现 > 商品」标签
2. 点击「+ 新商品」
3. 填写商品信息：标题、价格、品类
4. 点击「创建」

### 5.2 AI 商品描述

1. 在商品卡片点击「🤖 AI描述」
2. 选择目标平台风格（小红书/淘宝/Instagram/Etsy 等）
3. 点击「生成 AI 描述」
4. 复制生成的描述文案

### 5.3 生成 Verified 徽章

Original认证徽章可嵌入到您的商品页或网站，证明内容的原创性：

1. 进入「变现 > 徽章」标签
2. 选择目标商品
3. 点击「生成 Verified 徽章」
4. 下载 SVG / PNG 格式徽章
5. 或复制 HTML 嵌入代码到网站

### 5.4 产品 Feed 导出

将产品数据导出为各平台兼容格式：

1. 进入「变现 > Feed导出」标签
2. 选择目标平台：
   - **Universal**: 标准 JSON 格式
   - **Google**: Google Merchant Center XML
   - **Shopify**: Shopify 产品 CSV
   - **旺店通**: 旺店通 ERP 导入格式
   - **聚水潭**: 聚水潭 ERP 导入格式
   - **万里牛**: 万里牛 ERP 导入格式
3. 可选：按品类筛选
4. 点击「导出 Feed」
5. 下载 JSON 文件

### 5.5 收入管理

1. 进入「变现 > 收入」标签
2. 手动记录：点击「+ 记录收入」，填写平台/金额/日期
3. CSV 导入：点击「📥 导入CSV」，支持淘宝/抖音等格式自动检测
4. 查看汇总统计和平台分布

## 6. 变现引擎

### 6.1 变现路径

OriStudio 支持五条变现路径：

| 路径 | 说明 | 典型平台 |
|------|------|---------|
| POD (按需打印) | 上传设计到POD平台自动生产 | Printful, Redbubble, 印鸽 |
| 众筹 | 发起产品众筹项目 | 摩点, Kickstarter |
| IP 授权 | 出售设计授权许可 | Creative Fabrica, Creative Market |
| 定制制造 | 对接工厂小批量定制 | 1688, 定制供应商 |
| 数字产品 | 出售数字素材/模板/字体 | Gumroad, Etsy, Envato |

### 6.2 合作伙伴管理

在「供应链 > 合作伙伴」管理制造商和供应商：

- 添加工厂信息（名称、联系方式、产品类型、起订量）
- 记录制造能力和价格区间
- 工厂比价工具：输入品类和数量，自动对比报价

### 6.3 订单管理

- 创建生产订单（关联合作伙伴和产品）
- 跟踪订单状态（草稿→确认→生产中→已发货→完成）
- 样品管理：请求样品、确认样品

### 6.4 众筹项目

- 创建众筹项目（设置目标金额、奖励档位）
- 使用档位模板快速搭建众筹结构
- 目标金额计算器
- 导出项目报表

## 7. 侵权监控

### 7.1 创建扫描任务

1. 进入「监控」页面
2. 点击「+ 新建扫描」
3. 选择扫描源作品
4. 选择扫描范围（百度图片/Google图片/全平台）
5. 创建任务后等待扫描完成

### 7.2 查看扫描结果

- 匹配的侵权链接列表
- 相似度评分和详情
- 一键生成侵权报告
- 支持导出扫描结果

## 8. 系统设置

### 8.1 常规设置

- 站点名称和描述
- 默认语言和时区
- 上传文件大小限制

### 8.2 通知设置

- **邮件通知**: 配置 SMTP 服务器，测试发送
- **微信模板消息**: 配置公众号 AppID/AppSecret，发送模板消息通知

### 8.3 数据管理

- **备份**: 手动/定时备份数据库，支持 AES 加密
- **恢复**: 从备份恢复数据
- **导出**: 导出所有数据（JSON/CSV格式）
- **清除**: 清除所有本地数据（危险操作，需要确认）

### 8.4 安全设置

- 修改密码（含密码强度检测）
- 上传头像
- 邮箱验证
- TOTP 双因素认证（即将上线）
- 账号注销

### 8.5 API 密钥

在设置页面管理第三方 API 密钥：

- 百度视觉搜索 API
- Google Vision API
- Copyscape API
- 版权家 / 蚂蚁链 / 知信链 API
- 打印派 / Redbubble API（可选）

## 9. 常见问题 (FAQ)

### 9.1 启动问题

**Q: 启动后端报错 "ModuleNotFoundError: No module named 'xxx'"**

A: 请确保已安装依赖：
```bash
cd backend
pip install -r requirements.txt
```

**Q: 前端访问后端 API 报 CORS 错误**

A: 检查后端是否已启动，前端开发服务器会自动代理到 `http://localhost:8765`。

**Q: Ollama 连接失败**

A: 确保 Ollama 已安装并运行：
```bash
ollama serve
```
在不使用 AI 功能时，系统会自动降级为模板模式，不影响基本功能。

### 9.2 功能问题

**Q: 版权存证需要付费吗？**

A: 各平台收费标准不同：
- 蚂蚁链：免费额度（通常每月 5-10 次）
- 版权家：按次收费
- 知信链：联盟成员免费

具体以各平台最新政策为准。

**Q: Feed 导出格式不完整怎么办？**

A: 确保产品信息完整（标题、描述、价格、品类）。不同平台的导出格式会自动适配字段映射。

**Q: CSV 导入收入识别失败？**

A: 系统支持自动检测淘宝/抖音/通用格式的 CSV。确保 CSV 包含必要的列：金额、日期、平台。如果不确定格式，先导出示例 CSV 作为模板。

**Q: 产品能同时在多个 POD 平台发布吗？**

A: 可以。OriStudio 支持同时管理多个 POD 平台的产品：
- Printful: API 集成
- Redbubble: CSV 模板 + 手动上传
- Society6 / Gelato / Zazzle / Spring: 规格参考 + mock 适配
- 印鸽 / 云打 / 定制链 / 山隐: 中国 POD 规格参考

### 9.3 数据安全

**Q: 我的数据存储在哪里？**

A: 所有数据存储在本地 `data/` 目录下：
- `data/oristudio.db`: SQLite 数据库
- `data/workspace/`: 上传的作品文件
- `data/backups/`: 备份文件

**Q: 如何迁移数据到另一台电脑？**

A:
1. 在旧电脑执行「设置 > 数据管理 > 导出所有数据」
2. 复制整个 `data/` 目录到新电脑
3. 在新电脑启动 OriStudio

**Q: 敏感信息如何保护？**

A: API 密钥和密码等敏感信息使用 AES-256 加密存储。备份文件可选择 AES-256-GCM 加密。

### 9.4 性能与限制

**Q: 支持的最大文件大小？**

A:
- 作品上传: 默认 50MB，可在设置中调整
- 头像上传: 最大 5MB
- CSV 导入: 建议单次不超过 1000 条记录

**Q: 能处理多少产品？**

A: 系统设计支持数万级别产品（SQLite 单文件数据库），建议定期备份。对于更大规模使用，可考虑迁移到 PostgreSQL。
