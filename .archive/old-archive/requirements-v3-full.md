## 二、需求的"最大公约数"——所有创作者共性的需求

### 全类型创作者统一工作流

```
创作 → 保护 → 产品化 → 分发推广 → 变现收入
  ↑_________________________________|
```

### 共性需求矩阵

| 阶段 | 所有类型共有需求 | 覆盖的创作者 |
|------|-----------------|-------------|
| **资产管理** | 文件导入、元数据编辑、版本管理、标签分类、作品搜索、权利声明、许可证选择 | 全部6种 |
| **存证保护** | 哈希摘要、时间戳存证、证书生成、区块链锚定 | 全部6种 |
| **侵权追踪** | 内容相似度检测、结果审核、证据打包 | 插画/摄影/视频/音乐 |
| **IP登记** | 版权/商标/外观设计登记指引、智能助手、费用计算 | 全部6种 |
| **产品化** | 将IP应用于实体/数字产品、效果预览 | 全部6种（产品形态不同） |
| **分发推广** | 多平台发布、AI文案生成、格式适配 | 全部6种 |
| **变现追踪** | 多渠道收入汇总、趋势分析 | 全部6种 |

## 三、各类型差异需求分析

### 3.1 作品文件类型差异

| 创作类型 | 文件格式 | 元数据提取重点 | 预览方式 | 版本管理特点 |
|----------|----------|---------------|----------|-------------|
| 插画/AIGC | PNG/PSD/JPG | EXIF(尺寸/色彩空间)、图层信息 | 图片预览+放大 | 草图→线稿→上色→终稿 |
| 视频/动画 | MP4/MOV | 分辨率/帧率/编码/时长 | 视频播放器 | 脚本→分镜→粗剪→精剪→成片 |
| 手工艺品 | JPEG(拍摄) | 材质/尺寸/重量、制作工艺 | 多角度照片 | 样品→成品→系列化 |
| 摄影 | RAW/JPEG | EXIF(相机/镜头/光圈/ISO/GPS)、调色参数 | 高清预览+对比 | 原片→精选→精修 |
| 音乐/音频 | MP3/WAV | 采样率/比特率/BPM/调性 | 音频播放器 | 灵感→编曲→录音→混音→母带 |
| 文字 | TXT/MD/PDF | 字数/章节/编码 | 文本阅读器 | 大纲→初稿→修订→终稿 |

### 3.2 变现产品形态差异

| 创作类型 | 产品形态 | 变现渠道 | 对接的第三方 |
|----------|----------|----------|-------------|
| 插画/AIGC | T恤/卫衣/手机壳/杯子/海报/贴纸/帆布袋 | Printful/Redbubble/印鸽/Society6/Etsy | POD平台API |
| 视频/动画 | 付费课程/会员专享/品牌商单/衍生品 | B站创作激励/YouTube Partner/抖音 | 各平台创作者后台 |
| 手工艺品 | 实物成品/DIY套件/制作教程 | Etsy/淘宝/线下市集 | Etsy API/物流API |
| 摄影 | 数字下载/艺术微喷/预设包/图库 | Shutterstock/500px/图虫/Adobe Stock | 图库平台API |
| 音乐/音频 | 数字专辑/采样包/商用授权/在线课程 | Spotify/QQ音乐/Apple Music | DistroKid/TuneCore |
| 文字 | 实体书/电子书/有声书/影视改编权 | Kindle KDP/起点/晋江/微信读书 | KDP API/版权交易平台 |

### 3.3 保护手段差异

| 创作类型 | 核心保护手段 | 侵权检测方式 | 维权工具 |
|----------|-------------|-------------|---------|
| 插画/AIGC | SHA-256哈希 + 区块链存证 | 百度识图/Google Vision 以图搜图 | DMCA通知/证据包 |
| 视频/动画 | 视频指纹/数字水印 | YouTube Content ID | Content ID 申诉 |
| 手工艺品 | 设计图纸存证 + 外观专利 | 平台侵权举报 | Etsy举报/DMCA |
| 摄影 | EXIF+哈希 + 可见/不可见水印 | 以图搜图/反向图像搜索 | DMCA通知/版权索赔 |
| 音乐/音频 | 音频指纹 + PRO注册 | Shazam/Content ID | DMCA通知/版税追索 |
| 文字 | 创作时间戳 + 文本哈希 | 查重系统/抄袭检测 | 平台侵权举报 |

### 3.4 分发渠道差异

| 创作类型 | 核心分发平台 | 内容格式要求 | 发布时间策略 |
|----------|-------------|-------------|-------------|
| 插画/AIGC | 小红书/站酷/B站/微博 | 图片+文案(小红书需要实拍感) | 工作日午间/晚间 |
| 视频/动画 | B站/抖音/YouTube | 视频(16:9/9:16两种比例)+标题封面 | 黄金时段(18:00-20:00) |
| 手工艺品 | 小红书/淘宝/Etsy | 多角度产品图+参数说明+视频 | 周末购物高峰期 |
| 摄影 | 500px/Instagram/图虫 | 高清大图+拍摄参数+标签 | 早晚通勤时段 |
| 音乐/音频 | 网易云/QQ音乐/Spotify | 音频文件+封面图+歌词 | 周五(新歌发布日) |
| 文字 | 起点/知乎/微信公众号 | 章节/段落+标题+封面图 | 定期更新(日更/周更) |

## 五、系统功能需求整合 — 面向全类型

### 01 作品管理（全类型）

| 功能 | 说明 |
|------|------|
| **多格式导入** | 图片(PNG/JPG/PSD/RAW)、视频(MP4/MOV)、音频(MP3/WAV)、文档(TXT/MD/PDF)、3D模型(STL/OBJ) |
| **智能元数据提取** | 图片EXIF、视频编码/帧率、音频BPM/调性、文档字数/章节 |
| **统一版本管理** | 每种类型有自己的过程阶段：插画(草图→线稿→终稿)、视频(粗剪→精剪→成片)、音乐(编曲→混音→母带)、文字(大纲→初稿→终稿) |
| **权利信息面板** | 作者署名、版权年份、许可证选择、权利声明 |
| **项目/系列管理** | 按系列/项目分组（山海经系列/2026春季系列） |

### 02 存证确权（全类型）

| 功能 | 说明 |
|------|------|
| **即时哈希存证** | 导入时自动SHA-256，记录时间戳 |
| **多平台锚定** | 版权家(DCI) / 蚂蚁链 / 至信链 |
| **PDF证书** | 包含作品缩略图、哈希、时间戳、平台证明、QR验证码 |
| **批量存证** | 一次选择多件作品批量存证 |

### 03 侵权监测（图片/视频/音乐）

| 功能 | 说明 |
|------|------|
| **以图搜图** | 百度识图 + Google Vision |
| **视频指纹** | 预览片段提取视频指纹 |
| **音频指纹** | 提取音频特征与平台对比 |
| **文本查重** | 与目标URL内容对比 |
| **结果审核** | 确认侵权/误报/白名单 |
| **证据包** | 自动打包截图+URL+时间戳+哈希 |
| **定时扫描** | Celery Beat 每周自动扫描 |

### 04 IP登记（全类型）

| 功能 | 说明 |
|------|------|
| **多辖区指引** | 中国/美国/欧盟/WIPO/日本/韩国 |
| **多IP类型** | 著作权/商标/外观设计/专利 |
| **智能预填** | 从作品信息/Profile自动填80%字段 |
| **费用计算器** | 多辖区多品类费用对比 |
| **登记记录** | 手动登记+进度追踪+续展提醒 |
| **类别推荐** | 按标签/描述推荐尼斯分类 |

### 05 创作变现引擎（★核心模块）

| 子模块 | 功能 | 覆盖创作者 |
|--------|------|-----------|
| **产品设计器** | 选设计 → 选产品(60+品类) → 效果预览 → 规格适配 → 定价 → 创建产品 | 全部 |

| **实物销售** | Etsy/淘宝/线下市集 产品管理 | 手工艺人/插画 |
| **图库销售** | Shutterstock/500px/图虫 作品上架 | 摄影/AIGC |
| **音乐发行** | DistroKid/TuneCore 数字发行 | 音乐人 |
| **内容变现** | B站创作激励/YouTube/付费专栏 | 视频/文字 |
| **众筹管理** | 摩点/Kickstarter 众筹项目创建+进度 | 插画/手工艺 |
| **授权管理** | IP授权记录(授权对象/用途/期限/费用) | 全部 |
| **收入汇总** | 全渠道收入仪表盘+趋势分析 | 全部 |

### 06 分发推广中心（★核心模块）

## 七、MVP实施计划
