# OriStudio UX 交互设计规范

> **日期**: 2026-06-12 | **基于**: 9 角色评估报告 + 7 模块 v3 设计文档
> **目标**: 确保新手 3 分钟内进入可用状态，消除术语障碍，统一交互模式

---

## 一、首次使用引导流程 (Onboarding Wizard)

3 步引导，选择结果持久化到 `users.creator_type`、`system_settings.onboarding_completed`。

### Step 1: 选择创作者类型

3 列 x2 行卡片网格。v1 版本插画师卡片视觉突出（大尺寸 + 闪光渐变边框 + `推荐` 徽章），其余诚实标注能力状态。

选择后系统自动调整：作品过程阶段列表 / 默认变现路径 / 推荐分发平台（前 3）/ 产品品类过滤

### Step 2: 导入第一批作品

大面积友好拖拽区域 + 支持格式图标说明 + "跳过，稍后导入"链接。拖拽文件夹时自动按子文件夹名称创建项目组。

### Step 3: 快速上手引导

成功仪式感 + 简化 3 步路径可视化（🎨 管理作品 → 🛡️ 保护版权 → 💰 开始变现） + "开始使用"按钮 → 进入首页

## 二、空状态页面设计

**总原则**：空状态 = "下一步做什么" 的标准答案。每个空状态：图标 + 标题 + 描述 + 主 CTA + 可选副 CTA + 提示列表。

| 场景 | 图标 | 标题 | CTA |
|------|------|------|-----|
| 空作品库 | 📁 | 还没有作品 | 导入第一件作品 |
| 空仪表盘 | 🚀 | 开始你的创作之旅 | 导入第一件作品 |
| 空产品列表 | 🎨 | 将 IP 变成产品 | 设计第一个产品 |
| 空收入 | 💰 | 收入数据将自动汇总 | 设计产品 / 探索变现 |
| 空存证记录 | 🛡️ | 保护你的作品 | 快速存证 |
| 空监测结果 | 🔍 | 暂无侵权监测结果 | 开启监测 |
| 空 IP 登记 | 📋 | 暂无 IP 登记记录 | 登记指引 |
| 空合作伙伴 | 👥 | 暂无合作伙伴 | 添加伙伴 |
| 空通知 | 🔔 | 暂无通知 | — |
| 搜索无结果 | 🔍 | 未找到匹配结果 | 清除筛选 |

## 三、术语优化方案

**三层策略**：Layer 1 用户界面（白话 + 结果） / Layer 2 hover 提示（简短解释） / Layer 3 点击展开（技术细节）

### 核心术语对照表

| 当前术语 | 优化后 (Layer 1) | Layer 2 (hover) |
|---------|-----------------|-----------------|
| SHA-256 哈希 | **"已存证 ✅ YYYY-MM-DD"** | "数字指纹已锚定到区块链" |
| EXIF 信息 | **"拍摄参数"**（摄影）/ **"图片信息"**（其他） | "相机型号、光圈、快门等" |
| DPI | **"清晰度"** / **"印刷质量 300"** | "每英寸点数，影响印刷精细度" |
| POD | **"一键印品"** | "按需印制，零库存，有订单才生产" |
| C2PA 内容凭证 | **"AI 生成标注"** | "Content Credentials 标准来源信息" |
| WIPO | **"国际商标"** + "马德里体系" | "世界知识产权组织马德里体系" |
| 尼斯分类 | **"商标类别推荐"** | "尼斯协定商品/服务国际分类" |
| 洛迦诺分类 | **"外观设计类别"** | "洛迦诺协定工业品外观设计分类" |
| Canvas mockup | **"产品效果预览"** | — |

## 四、交互模式规范

### 4.1 列表页统一模式

搜索框 (debounce 300ms) + 筛选器（类型 + 状态） + 视图切换（列表/网格） + 批量操作栏（选中 ≥1 出现） + 分页 + 空状态按规范渲染 + 加载 Skeleton + hover 快捷操作

### 4.2 详情页统一模式

← 返回列表 + 操作按钮 + 预览区（40% 宽） + 信息面板（60% 宽：基本信息/属性/关联/版本历史） + 底栏 CTA

### 4.3 创建/向导流程统一模式

步骤指示器（已完成可回退） + 底部导航（← 上一步/下一步→） + 保存草稿（Pinia store，支持恢复） + 关闭确认 Modal

### 4.4 加载状态规范（永不空白）

- 页面首次：全页 Skeleton
- 列表数据：8 行 Skeleton
- 文件导入：进度条 + "正在导入 12/47"
- AI 生成：打字动画 "AI 正在为你生成..."
- 存证提交：4 步状态流转

### 4.5 错误状态规范

不说"出错了"，说"发生了什么 + 你可以怎么做"。每个错误场景：具体描述 + 可能原因 + 重试按钮

### 4.6 通知反馈规范

| 类型 | 表现 |
|------|------|
| 操作成功 | Toast 顶部居中 绿色 2 秒 |
| 操作失败 | Toast 顶部居中 红色 4 秒 |
| 操作进行中 | Toast 蓝色 + Spinner 直到完成 |
| 警告 | Banner 内容区顶部 手动关闭 |
| 确认 | Modal 页面居中 用户操作 |
| 后台完成 | Sidebar Badge teal 圆点 |

## 五、移动端响应式适配

### 断点

| 断点 | 侧边栏 | 内容区 |
|------|--------|--------|
| >=1024px 桌面 | 固定 240px | 剩余 |
| 768-1023px 紧凑桌面 | 汉堡菜单弹出 | 全宽 |
| 480-767px 平板 | 底部 Tab | 全宽单列 |
| <480px 手机 | 底部 Tab | 全宽单列 |

### 导航

- 侧边栏 9 入口 → 底部 5 Tab：⌂ 首页 / 📂 作品 / 🛡️ 保护 / 💰 变现 / ⚙ 我的
- 列表：表格 → 单列卡片
- 弹窗：Modal → 全屏 Sheet（顶部指示条下拉关闭）
- 触摸目标：>=44x44px

## 六、实施优先级

1. Onboarding Wizard 3 步引导
2. 5 个核心空状态（作品库、仪表盘、产品、收入、存证）
3. 9 个术语替换（Layer 1 层面）

7. 详情页统一布局
8. 创建向导步骤指示器 + 草稿保存
9. 确认操作 Modal
10. 错误状态统一处理

### Phase 3 (P2) — 完整体验

11. 移动端底部 Tab 导航
12. 移动端卡片视图 + Sheet
13. 键盘导航 + 焦点管理
14. 暗色模式
15. 离线检测

### Phase 4 (P3) — 锦上添花

16. 移动端产品设计器简化版
17. AI 文案打字动画
18. 存证步骤 4 步可视化
19. 首次 IP 登记免责声明
20. 浏览器通知 API 集成
