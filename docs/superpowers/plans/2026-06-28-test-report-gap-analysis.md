## 未覆盖问题汇总 (11 项)

以下问题在当前所有计划中均无覆盖，需要补充:

### U1: 登记记录台账页面 (P1)
- 测试报告用例: 模块四-用例3
- 需求: 展示历史提交登记申请、审核状态、证书编号、申请时间、操作(查看详情/撤回/补材料)
- 后端: IPRegistration 模型已存在，需要列表 API + 前端页面
- 前端: `IprRecordListView.vue`

### U2: IP 资产总览页面 (P1)
- 测试报告用例: 模块四-用例4
- 需求: 汇总名下所有已确权著作权/商标/专利资产台账，分类统计、筛选、导出
- 后端: 需要聚合查询 API
- 前端: `IpAssetDashboardView.vue`

### U3: 导入作品功能增强 (P1)
- 测试报告用例: 模块二-用例2
- 需求: 下拉式导入菜单，区分本地上传/批量导入/第三方平台同步，拖拽支持，格式大小说明，上传进度，失败重试
- 前端: 增强 `ImportWorkModal.vue` 或新建 `ImportWorkDrawer.vue`

### U4: 专利模块英文字段汉化 (P1)
- 测试报告用例: 模块九-1
- 需求: 将 `application`/`registration`/`annual_1_3` 等英文变量翻译为中文
- 文件: 搜索 IPR 相关前端组件中的硬编码英文文本

### U5: 多国地区内容补充 (P2)
- 测试报告用例: 模块六
- 需求: 美/欧/WIPO/日韩知识产权登记指引内容，未完成地区置灰标注"建设中"
- 前端: IprView.vue 中地区标签逻辑 + 各国内容数据

### U6: 智能助手自动填表 (P2)
- 测试报告用例: 模块四-用例2
- 需求: 选择创意资产一键提取信息、自动填表，补充功能说明文案
- 前端: `IpSmartAssistant.vue` 组件

### U7: 费用计算器 (P2)
- 测试报告用例: 模块四-用例5
- 需求: 选择登记类型/数量，自动核算官方申请费用，展示明细和总金额
- 前端: `IpFeeCalculator.vue` 组件

### U8: 费用信息清晰度 (P2)
- 测试报告用例: 模块七-6, 模块八-2
- 需求: 补充著作权费用明细、商标续展费/驳回复审费、加急通道说明
- 前端: IprView.vue 中费用相关文案

### U9: 预填功能触发入口 (P2)
- 测试报告用例: 模块七-1, 模块八-4
- 需求: 著作权/商标登记指引中增加"一键预填"按钮
- 前端: IprView.vue 中各指引卡片

### U10: 消息通知完善 (P1)
- 测试报告用例: 模块二-用例3
- 需求: 未读角标、消息分类(确权/维权/收益/系统公告)、已读管理
- 后端: Notification 模型已存在，需要分类和角标 API
- 前端: 增强通知铃铛组件

### U11: 深色模式持久化 (P3)
- 测试报告用例: 模块二-用例1
- 需求: 主题切换配置本地缓存，刷新不重置，支持跟随系统
- 前端: AppTopbar.vue 或新建 theme store
